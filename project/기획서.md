
## 갈만해?

실시간 날씨와 인구혼잡도를 계산하여 서울 주요 장소들의 외출하기 좋은 정도를 보여주는 지도제공 서비스.

## 용어사전

외출도 : 장소의 날씨와 혼잡도를 기반으로 외출 나갈만한 정도를 한 문장으로 표현한 것. (ex. 외출하기 좋아요)

날씨 문장 : 장소의 온도와 강수확률을 한 문장으로 표현한 것 (ex. 맑고 청명해요)

혼잡도 문장 : 장소의 실시간 인구수와 단위면적 당 인구수를 한 문장으로 표현한 것(ex. 여유로움)

장소 정보: 외출도, 날씨, 혼잡도를 한번에 보여주는 창.

모든 장소 : 장소정보를 보여줄 수 있는 서울의 115곳 인구밀집지역

## 핵심 기능

- 지도에 모든 장소를 핀으로 보여준다.
- 장소 핀을 선택할 시 화면 오른쪽에 장소정보를 보여준다
    - 외출도를 보여준다.
    - 날씨 문장을 보여준다
    - 날씨 세부사항으로 구체적인 온도와 강수확률을 보여준다
    - 혼잡도 문장을 보여준다
    - 혼잡도 세부사항으로 실시간 인구와 단위면적 당 인구수를 보여준다
- 장소 검색을 할 수 있다

## 세부 기능

- 지도
    - 처음 접속시 사용자의 현재 위치를 기반으로 확대된 지도를 보여준다
        - 위치 정보 공유 거절 시 시청역을 기본 위치로 한다
    - 기본 줌은 300m
- 장소  (점수 및 문장 상세는 밑의 표로 정리)
    - 장소 데이터는 1시간 간격으로 업데이트한다.
    - 날씨 문장은 온도와 강수확률을 기반으로 'A하고 B해요'로 표현한다. 각 상태에 점수가 부여된다 (ex. 춥고 비와요)
    - 혼잡도 문장은 단위 면적당 인구수로 '~해요'로 표현한다. 각 상태에 점수가 부여된다 (ex. 여유로워요)
    - 외출도는 날씨와 혼잡도의 점수를 종합해 한 문장으로 표현한다. (ex.외출하기 좋아요)
    - 외출도의 단계에 따라 장소핀의 색깔이 황금색- 은색 - 동색으로 바뀐다. (변동 가능성 있음)
- 검색
    - 유저가 입력하는 글자에 맞춰 자동완성으로 장소 이름후보를 보여준다
    - 엔터를 입력하거나 '갈만해?'버튼을 누르면 지도가 이동하고 오른쪽에 장소 정보가 뜬다.
    - 없는 장소를 입력하면 가장 가까운 장소를 보여준다(옵션)

(각 문장 상세와 점수는 변동 가능성이 있다)

- 날씨 문장 상세

사용하는 데이터 : 기상청 단기예보

| 기온    | 앞 문장 | 점수  | 강수확률   | 뒷 문장 | 점수  |
| ----- | ---- | --- | ------ | ---- | --- |
| ~ 5도  | 춥고   | 5   | ~ 10 % | 맑아요  | 40  |
| ~ 10도 | 쌀쌀하고 | 20  | ~ 50 % | 흐려요  | 20  |
| ~ 20도 | 시원하고 | 40  | ~100 % | 비와요  | 5   |
| ~ 30도 | 따뜻하고 | 20  |        |      |     |
| ~ 40도 | 덥고   | 5   |        |      |     |

- 혼잡도 문장 상세

사용하는 데이터 : 서울시 실시간 인구데이터
장소 혼잡도 지표는 서울시 실시간 인구데이터에서 주는 '장소 혼잡도 지표'를 사용할 예정.
지표 추출 과정 : skt, kt에서 제공하는 데이터를 서울시에서 인구수를 장소의 면적으로 나누어서 4단계로 나누어 제공

| 장소 혼잡도 지표 | 문장     | 점수  |
| --------- | ------ | --- |
| 여유        | 여유로워요  | 50  |
| 보통        | 보통이에요  | 30  |
| 약간 붐빔     | 약간 붐벼요 | 15  |
| 붐빔        | 사람 많아요 | 5   |

- 외출도 상세

| 날씨 + 혼잡도 점수 | 문장             |
| ----------- | -------------- |
| ~ 40점       | 다른 장소를 찾아 볼까요? |
| ~ 70점       | 무난해요           |
| 70점 이상      | 외출하기 좋아요!      |
